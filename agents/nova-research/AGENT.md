# nova-research — Agent Specification

## Role
Web research and data lookup agent.
Handles information gathering, web search, news lookup, comparisons.

## Capabilities
- Web search via OpenClaw agent integration
- Prompt caching for repeated queries
- Local file search
- Data lookup and comparison

## Model
- Default: ollama/llama3.2
- Fallback: anthropic/claude-sonnet-4-6
